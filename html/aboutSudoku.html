Objective:
Sudoku is a popular logic-based number placement puzzle. The objective is to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 subgrids that compose the grid contain all of the digits from 1 to 9. The puzzle initially starts with a partially filled grid, and the player's task is to fill in the missing numbers.
Feedback:
The game is made so if you make a mistake, and a number appears twice in a row, subgrid, or column, it ligths up red.
Winning the game:
Winning in Sudoku is achieved when the player successfully fills in all the empty cells in the 9x9 grid according to the rules. This means that each row, column, and 3x3 subgrid contains all the numbers from 1 to 9 without any repetition.
